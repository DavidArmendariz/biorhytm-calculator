## Ionic commands

- Serve

```
ionic serve
```

- Make a build
- Add android support

```
ionic build
ionic cap add android
ionic cap open android
```

To sync changes:

```
ionic cap sync android
```

- To use live reload and add the server automatically in `capacitor.config.json`

```
ionic cap run android --livereload --external
```
